# Journey to the Center of Hawkthorne — Community 3×20

Tributo fan-made a **Community** S3E20 *"Digital Estate Planning"*: un platformer 8-bit
completo in un singolo file HTML, con gli sprite autentici di
[Project Hawkthorne](https://github.com/hawkthorne/hawkthorne-journey) (il gioco
open-source creato dalla community ricalcando la puntata).

**Gioca:** apri `index.html` con un doppio click, oppure dalla versione online (GitHub Pages).

## Caratteristiche
- 20 personaggi di Greendale giocabili (7 gratis, gli altri si sbloccano a monete; Gilbert finendo il gioco)
- 9 costumi originali (tasto **C** nella selezione)
- **Co-op locale a 2 giocatori** (P2 entra premendo **F**) e **co-op online P2P** via WebRTC (tasto **O** al titolo, scambio di 2 codici, senza server)
- **Joystick supportati** (Gamepad API: 2 pad = 2 giocatori)
- Super-poteri caricati a "Meraviglia": Discorso Winger, Brittapocalisse, **JAILBREAK di Abed**, Laser Lotus liv. 6 di Pierce…
- Negozio di Hilda con benedizioni casuali (stile Hades), forgia della Lancia-Lava, casse, combo, spiriti di Cornelius, **Modalità Incubo** post-game
- 5 mondi: sala d'attesa Hawthorne (hub), radura, villaggio dei contadini, caverne, trono di Cornelius — con l'esercito di bambini di Abed nel finale

## Controlli
| | P1 | P2 |
|---|---|---|
| Muovi | frecce | WASD |
| Salto | Z / Spazio | W |
| Attacco | X | F |
| Abilità | C | G |
| Super | V | H |
| Negozio/Forgia | GIÙ | S |

`P` pausa · `M` audio · `F2` effetto CRT · `O` co-op online (dal titolo)
